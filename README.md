💡 Key Insights
Home loans account for the highest amount issued, while other purposes show the lowest.

Unemployed applicants face the highest default rate (3.39%), whereas full-time employees show the lowest (2.86%).

Adults consistently borrow higher amounts across the age groups.

Default rates were highest in 2016 (11.75%) and lowest in 2014 and 2017.

Higher credit scores correlate with more conservative loan amounts.

📌 Project Purpose
This Power BI report was developed to study loan default patterns, allowing financial analysts and institutions to make better risk-based decisions. It also serves as a portfolio project to demonstrate expertise in data visualization, financial analytics, and storytelling with data.
